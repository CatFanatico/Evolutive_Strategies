# Evolutive_Strategies
This repository contains examples of genetic algorithms applied to straightforward optimization problems, utilizing various techniques and crossover operators such as UNDX. The implementations were written in Rust to leverage its execution speed and effectively handle the representation of solutions. Detailed reports in PDF format are also included

CONTENTS:
 - Queens_Perms: Contains a genetic algorithm that solves the 8-Queens problem using a vector-based representation (with permutation-inducing operators).

 - Queens_matrix: Analogous to the previous directory, but employs a matrix-based representation for the solutions.

 - rastrigin: Features a genetic algorithm tasked with optimizing the Rastrigin function.

The codebase, updated as of December 31, 2025, was originally developed to fulfill Master's level coursework requirements at the Instituto de Investigaciones en Inteligencia Artificial. Future updates and code optimizations will be implemented subject to the author's availability. Additionally, reports written in Spanish are included, detailing some of the results obtained using these methods
